# Data Analytics Internship - Task 1

## Project Overview
This repository contains the deliverables for Task 1 of my Data Analytics Internship. The goal of this task was to clean a raw e-commerce dataset by managing missing values, eliminating duplicates, and standardizing data types.
## Deliverables
* `Dataset for Data Analytics - Sheet1.csv`: The raw dataset.
* `Task_1_Data_Cleaning.ipynb`: The Jupyter Notebook containing the data preparation python code.
* `Cleaned_Dataset_Task1.csv`: The final, cleaned output dataset.
## Key Actions Taken
* Handled `309` missing fields in the `CouponCode` column by replacing them with `'None'`.
* Confirmed `0` duplicates across rows.
* Cast the `Date` column into a standardized datetime format and enforced numeric schemas on price and quantity variables.
